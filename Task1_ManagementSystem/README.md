Student Management System – Python (CSV Based)

This is a simple command-line Student Management System built using Python. 
It supports adding, viewing, updating, and deleting student records using a CSV file.

Features:
1. Add Student – Stores RollNo, Name, and Class
2. View Students – Displays all saved records
3. Update Student – Modify existing student details
4. Delete Student – Remove a student by RollNo
5. Data stored in students.csv
6. Uses try-except for basic error handling
7. Fully modular, function-based code

Files Included:
• main.py – The full student management system code
• students.csv – Auto-created when records are added
• README.txt – Documentation and instructions

How to Run:
1. Install Python 3.
2. Open terminal/command prompt inside the project folder.
3. Run: 
       python main.py

CSV File:
• Automatically created on first run.
• Stores all student records.
