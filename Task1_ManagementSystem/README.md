
# 🎓 Student Management System - python (CSV based)

A simple Python-based **Student Management System** built as part of the **Advorix Python Internship 2026**. This project allows you to manage student records through a menu-driven program with full CRUD functionality.

## ✨ Features
- ➕ Add New Student  
- 📄 View All Students  
- 🔍 Search Student by ID / Name  
- ✏️ Update Student Details  
- ❌ Delete Student Record  
- 💾 Auto-Save Data using TXT/CSV  
- 📂 Clean, modular folder structure  

## 🛠️ Technologies Used
- 🐍 Python 3  
- 📁 File Handling  
- 📌 Functions & Modular Code
